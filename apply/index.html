[ FELLOWSHIP APPLICATION ]

About Code for Ghana

Do you want to empower citizens to better engage with their governments/communities? Code for Ghana is hiring the first set of fellows for its 2015 fellowship program, dedicated to help robust and independent media serve as a watchdog, holding government accountable and providing important information and analysis for citizens to more effectively engage in society and with government.

Code for Ghana will form a community of open-data experts and embed them into media organisations to bring open-data applications into Ghanaian newsrooms. This community of civic technologists will develop and use apps to analyse and interpret open data from sources such as the recently launched Ghana Open Data Initiative (GODI). The fellowship will enable a few hackers or “data fellows” with professional backgrounds in UI/UX Design, Data Analysis, Software Engineering and Research to work intensively within media organisations for six months. These data fellows will serve as catalysts and enablers who help create the internal systems and resources necessary to entrench data as a critical component of the Ghanaian newsroom culture.

User Interface & User Experience Fellow

Role

The ideal candidate will be required to collaborate with Software Engineers and Data Fellows to work on various data projects with a user-centric approach. Software that is developed as part of the project should be easy to understand and easy to use. We are looking for a user experience/user interface designer with strong interests and capabilities in the design and development of engaging user experiences. This individual excels at providing both highly analytical as well as highly creative ideas to a design engagement. A thorough understanding of data visualisation technologies and applications is a must.

Qualification

  • A rich background in UI/UX Design, Graphic Design or a relevant field.
  • In-depth knowledge of visualisation technologies, data visualisation, interaction design techniques for data visualisation.
  • Proficient with APIs and platforms for mobile and web-based data visualisation (eg JavaScript, HTML5, d3js, Raphaeljs, processing, tableau).
  • Ability to create visualisation graphics with Adobe Suite.
  • Passionate about user experience;
  • Ability to work in a small team.
  • Familiarity with developer tools and SDKs
  • Experience in facilitating user research and usability tests.
  • Experience in user interface patterns for mobile and web.

The ideal candidate will thrive at this intersection of software engineering and methodology research, possessing a passion for elegant, high performance, and scalable implementations of statistical/visual techniques that provide reliable insights for citizens using open data
