WHO WE ARE

Code for Ghana is part of the Code for Africa initiative which aims to use open data and technology to promote responsive, innovative and effective governance. Code for Ghana follows in the tradition of Code for Kenya and Code for South Africa by embedding software developers and data fellows in media organisations for a period of six months. During this period, the Data Fellows and Developers are expected to work with their host organisations on open data projects. Code for Ghana is jointly funded by the African Innovative News Challenge organised by the African Media Initiative and Making All Voices Count with support from the Webfoundation

Our Activities Include:

  • Build active Open Data ecosystem
  • Open up public data and provide easy access to citizens

  • Organise events to share knowledge, expertise and the value of open-data with the public.
  • Build civic apps and platforms that will promote citizen engagment
  • Work with CSOs, NGOs and Government organisations to bring meaning to their data-sets
